## Theme

---

## Docker

Runs webserver on localhost:8080 and mysql (mariadb) on localhost:3886

### Start docker
`docker-compose up`

---

## Build tools

### To install:

cd to the theme folder

`yarn` (or `npm install`)

### To build:
`yarn run build` (or `npm run build`)

### To develop:
`yarn run watch` (or `npm run watch`)
